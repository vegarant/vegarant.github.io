Code

This page contains an overview of the most noticeable code I have developed and which I believe others might have use for. If you are searching for code related to a specific paper, you may consult my Github page.

cslib

The cslib library is a large collection of scripts and functions related to compressive sensing and image reconstruction tasks in general. It has been developed in relation to the upcoming book Compressive imaging by B. Adcock and A. C. Hansen, published by Cambridge University Press. It is expected to be out in August 2021.

  • Language: Matlab
  • Code: https://bitbucket.org/vegarant/cslib/

fastwht

The fastwht library is a very fast (CPU) implementation of the fast Walsh-Hadamard transform. At the computational core it uses the fxt library. The library is interfaced with Matlab and Python, providing (1) a substantially faster implementation of the fwht function in Matlab and (2) an implementation of the Walsh-Hadamard transform in Python. It also includes an interface for the sequency ordered Walsh function.

  • Languages: C/C++, Matlab and Python
  • Code: https://bitbucket.org/vegarant/fastwht/

Continuous Walsh sampling and wavelet reconstruction

In many real world problems the samples one acquire comes from continuous integral transforms, rather than discrete measurements. If one does not take this into account when doing the reconstruction, it typically leads to noticeable artifacts in the reconstructed signal. This code takes the infinite dimensionality of continuous Walsh sampling and wavelet reconstruction into account and computes a vector-matrix multiplication between a finite section of the change of basis matrix in FFT-time. Note that this is not the same as a naive call fwht and wavedec functions in Matlab.

  • Language: Matlab
  • Code: https://bitbucket.org/vegarant/generalized-sampling-with-walsh-sampling/

spgl1

spgl1 is a wildly used optimization algorithm used for solving \[ \text{minimize}_{z \in \mathbb{C}^{N}} \| z\|_1 \quad\text{ subject to }\quad \|Az - b\|_2 \leq \epsilon \] This is a reimplementation of the spgl1 algorithm done in C++ rather than Matlab. It also includes a very slow implementation of a wavelet transform. This project was developed while writing my master thesis (2016), and have not been touched since.

  • Language: C++
  • Code: https://bitbucket.org/vegarant/spgl1/
